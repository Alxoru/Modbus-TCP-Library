## ✨ Особенности

- 🎯 **Максимальная простота** — понятный код, минимум абстракций
- 🚀 **Легковесность** — всего ~10 КБ, без внешних зависимостей
- 💻 **Кроссплатформенность** — работает на Windows, Linux, macOS
- 🔍 **Встроенный сканер сети** — автоматическое обнаружение устройств
- 📊 **Поддержка всех типов данных** — bool, int, float, 32-bit значения
- 🛡️ **Понятные исключения** — легкая отладка
- 📚 **Образовательный код** — идеально для изучения Modbus TCP
- 🔌 **Чистый TCP/IP** — не требует дополнительного оборудования
